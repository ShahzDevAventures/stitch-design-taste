# stitch-design-taste

A Claude Code skill that generates `DESIGN.md` files for Google Stitch screen generation. Translates anti-slop frontend engineering directives into Stitch's native semantic design language — descriptive, natural-language rules paired with precise values that Stitch's AI agent can interpret to produce premium, non-generic interfaces.

## What it does

- Outputs a `DESIGN.md` file that tells Google Stitch how to render screens with strict typography, calibrated color, asymmetric layouts, perpetual micro-motion, and hardware-accelerated performance
- Encodes a configurable design taste profile (variance, motion intensity, restraint) so the same skill produces different outputs for different brands without rewriting prompts
- Pairs with the rest of the Sirge design pipeline (CRO audit → copy → wireframe → Stitch generation → Figma → Liquid)

## Install

```bash
npx skills add ShahzDevAventures/stitch-design-taste
```
